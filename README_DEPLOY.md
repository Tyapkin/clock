# Инструкция по деплою на VPS

## Предварительные требования
1. **VPS** с установленными `docker` и `docker-compose`.
2. **SSH доступ** к серверу.

## Вариант 1: Автоматический деплой (скрипт)

1. Откройте файл `deploy.sh` и замените `USER_HOST` на ваш адрес (например, `root@192.168.1.1`).
2. Запустите скрипт:
   ```bash
   chmod +x deploy.sh
   ./deploy.sh
   ```

## Вариант 2: Ручной деплой

1. **Скопируйте файлы на сервер**:
   ```bash
   scp -r ./* user@your-vps-ip:/path/to/app
   ```

2. **Зайдите на сервер**:
   ```bash
   ssh user@your-vps-ip
   cd /path/to/app
   ```

3. **Запустите приложение**:
   ```bash
   docker-compose up -d --build
   ```

Приложение будет доступно на порту 8080 (или 80, если вы измените `docker-compose.yml`).
